# MTProxyL — Changelog

Все изменения проекта документируются в этом файле.

---

## v1.0.0 - v1.0.4 — 23.06.2026

### Исправления
- Исправлено: `tg_connect` перенесён из секции `[timeouts]` в `[general]` — telemt выдавал предупреждение `Unknown config key`
- Исправлено: tune whitelist для `tg_connect` теперь указывает на секцию `general`, а не `timeouts`
- Исправлено: при восстановлении бэкапа теперь загружаются настройки в память (`load_settings`, `load_secrets`, `load_nft_settings`) и предлагается перезапуск прокси
- Исправлено: при восстановлении бэкапа показываются восстановленные параметры (порт, домен, количество секретов)

### Новое
- Автовыбор свободного порта метрик при установке (диапазон 9090..9199) — не конфликтует с другими приложениями на сервере
- При запуске контейнера автоматическая проверка занятости metrics port с переключением на свободный
- Порт метрик отображается в меню настроек и доступен для изменения через пункт `[g]`
- NFT limiter автоматически устанавливает `nftables` если не установлен
- Режим эксперта полностью переработан:
  - Каталог всех параметров telemt config.toml с описаниями, default, hot-reload, диапазонами
  - Интерактивный выбор раздела → параметра → карточка с описанием → ввод с валидацией
  - Override сохраняются в отдельный файл `expert.conf` с максимальным приоритетом
  - Валидация override-файла перед применением
  - Возможность открыть override-файл в nano
  - Поддержка секций: general, general.modes, general.links, general.telemetry, network, server, server.conntrack_control, server.api, timeouts, censorship, censorship.tls_fetch, access
- Новые библиотеки: `expert_catalog.sh`, `expert_mode.sh`

### Улучшения
- Функция `uninstall()` больше не чистит глобальный Docker build cache (`docker builder prune` удалён)
- Вывод `docker stop/rm/rmi` подавлен — удаление чистое без мусорного вывода
- Удаляются образы и из локального registry, и из GHCR
- Значения по умолчанию изменены: `tg_connect=30`, `client_handshake=90`, `client_keepalive=120`
- README полностью обновлён: навигация с якорями, подробное описание CLI, разделы по NFT/iOS/Expert
